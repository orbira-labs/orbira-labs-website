# Orbira Labs Website

Orbira Labs çatı marka tanıtım sitesi.

## Yerel Görüntüleme

`index.html` dosyasını tarayıcıda açın veya basit bir HTTP sunucusu ile:

```bash
# Python 3
python3 -m http.server 8000

# Sonra tarayıcıda: http://localhost:8000
```

## Yapı

- `index.html` - Ana sayfa
- `css/style.css` - Stiller
